# Shakti – AI-Powered Women Safety App

Shakti is a real-time women safety application designed to provide immediate assistance during emergencies. The application enables users to trigger emergency alerts through voice commands and automatically shares critical information such as live location, audio recordings, and camera feeds with administrators or emergency contacts.

## Key Features

* Voice-activated emergency alert system
* Real-time GPS location tracking
* Background location and media services
* Live audio recording and camera streaming
* Android native integration using Java and MethodChannels
* Flask REST API backend for emergency data management
* Low-latency communication for rapid response
* Cross-platform mobile application built with Flutter

## Technologies Used

* Flutter & Dart
* Android (Java)
* Flask (Python)
* REST APIs
* GPS & Location Services
* MethodChannels
* JSON
* Background Services

## Project Objective

The primary goal of Shakti is to enhance personal safety by providing a fast, hands-free emergency response system that can instantly transmit critical information to help responders take timely action.
